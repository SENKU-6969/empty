Import from CollegeDoors

Sync your test results from demotests.collegedoors.com

One-click import from your test portal

CollegeDoors has no public API, but you can use our browser bookmarklet — a small script that reads the test table on the portal while you're logged in, and downloads a JSON file. Upload that file here and all your tests get added to EduMetrics instantly.

1

Add the bookmarklet

Drag the " Import to EduMetrics" button below to your browser's bookmarks bar.

2

Run it on the portal

Go to demotests.collegedoors.com/pages/cdmypage.php while logged in, then click the bookmarklet. A JSON file downloads automatically.

3

Upload & import here

Upload the downloaded collegedoors-tests.json in the section below. Preview and confirm, then your tests appear on the dashboard.
